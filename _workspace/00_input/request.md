# 사용자 요청

**날짜:** 2026-05-06
**원문:** "진행시켜!"
**컨텍스트:** Weather UI/UX Template Generator 하네스 초기 실행. 사용자가 별도 명세 없이 진행 지시.

## 해석된 요구사항

- **카테고리:** 다양성 우선 (특정 카테고리 명시 없음). 한국 기상 환경에 적합한 12~15개 카테고리.
- **시각 스타일:** 다양성 분배 (미니멀/글래스모피즘/일러스트/데이터 헤비/뉴모피즘/빈티지 균형).
- **지역 기준:** 한국 (PM2.5, AQI 한국 기준, 한국어 라벨).
- **산출 개수:** 15개 템플릿 + index.html 카탈로그.
- **필수 카테고리 (도메인 전문가가 최종 결정):**
  1. Current weather (현재 날씨)
  2. Hourly forecast (시간별 예보)
  3. Daily forecast (주간 예보)
  4. Severe alert (특보/경보)
  5. Air quality (대기질, PM2.5/PM10/AQI)
  6. UV index (자외선)
  7. Wind (풍향/풍속, 윈드로즈)
  8. Precipitation (강수량/강수확률)
  9. Comparison (오늘 vs 어제, 어제 대비)
  10. Sunrise/Sunset (일출일몰)
  11. Climate infographic (기후 인포그래픽)
  12. Pollen / 황사 (계절성)
  13. Marine / Surf (해양/파도) — 옵션
  14. Outdoor activity (야외활동 적합도) — 옵션
  15. Travel weather (여행지 비교) — 옵션

## 페르소나 (가설)

- **출근 직장인:** 통근 시간대 강수·체감온도·미세먼지 중심
- **야외활동가:** UV·풍속·강수확률·일몰
- **농업인/조경:** 강수·습도·서리·일조시간
- **알레르기 환자:** 황사·꽃가루·PM2.5
- **여행자:** 다지역 비교·시간대별 변화

## 품질 게이트

- 모든 HTML은 더블클릭 즉시 렌더링 (Tailwind CDN + 인라인 SVG/Chart 라이브러리 CDN 허용).
- WCAG AA 색대비, 반응형(360px~1440px), 콘솔 에러 0.
- 한국어 라벨 + 단위 (°C, m/s, mm, ㎍/㎥, J/㎡).
