# data-platform

## 하네스: Weather UI/UX Template Generator

**목표:** 날씨 지표·인포그래픽·대시보드를 위한 다양한 UI/UX 템플릿(자체 완결 HTML)을 5인 에이전트 팀(도메인·UX·시각화·구현·QA)으로 생성한다.

**트리거:** 날씨/기상 관련 UI·위젯·인포그래픽·대시보드 작업 요청 시 `weather-template-orchestrator` 스킬을 사용하라. 단순 질문(예: "오늘 서울 날씨 알려줘")은 직접 응답 가능.

**산출물:**
- `templates/*.html` — 단일 HTML 자체 완결 템플릿 (Tailwind CDN + 인라인 SVG)
- `templates/index.html` — 모든 템플릿 카드 그리드 카탈로그
- `_workspace/` — 도메인 카탈로그·디자인 시스템·시각화 명세·QA 보고서 (감사 추적)
